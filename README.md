# HR Analytics Dashboard

## Project Overview
This project focuses on analyzing employee data to understand workforce trends, employee attrition, job satisfaction, and departmental performance using Power BI.

The dashboard provides valuable HR insights that help organizations improve employee retention, workforce planning, and decision-making.

## Objectives
- Analyze employee attrition trends
- Identify departments with high employee turnover
- Monitor employee demographics and age distribution
- Evaluate job satisfaction across different job roles
- Understand gender-based attrition patterns
- Generate interactive HR insights for better workforce management

## Tools Used
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization
- Dashboard Design

## Dataset Information
The dataset contains employee-related details including:
- Employee Count
- Age Group
- Gender
- Department
- Job Role
- Education Field
- Job Satisfaction Rating
- Attrition Status

## Dashboard Features
- Overall Employees KPI
- Active Employees Count
- Attrition Count & Attrition Rate
- Average Employee Age
- Department-wise Attrition Analysis
- Employee Distribution by Age Group
- Job Satisfaction Ratings by Job Role
- Education Field-wise Attrition
- Gender-based Attrition Analysis
- Interactive Visualizations and Filters

##  Key Insights
- Total Employees: 882
- Active Employees: 732
- Attrition Count: 150
- Attrition Rate: 17.01%
- R&D department has the highest attrition rate
- Most employees belong to the 25–34 age group
- Male employees show higher attrition compared to female employees
- Sales Executives and Research Scientists have high employee counts
- Life Sciences and Medical fields contribute most to attrition

##  Project Outcome
This HR Analytics dashboard transforms raw employee data into meaningful insights that help HR teams improve employee engagement, reduce attrition, and make data-driven workforce decisions.
